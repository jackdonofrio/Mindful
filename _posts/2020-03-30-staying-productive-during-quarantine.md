This is a list of things to do during the quanatine to stay productive.

+ Learn to code
+ Solve a puzzle
+ Exercise
+ Start a journal
+ Learn to meditate
+ Hike
+ Improve your breathing
+ Read/Listen to audiobooks
+ Start learning a new language
+ Write a story
+ Learn about digital minimalism
+ Watch lectures
+ Learn a cool skill
+ Take a cold shower
+ Start a new hobby
+ Take an online class
+ Learn about stoicism
+ Clean your room/home
+ Start a research project
+ Stay healthy

With everything you do, make sure you are setting goals. So if you're trying to exercise more, try setting a new personal mile record by running everyday.
